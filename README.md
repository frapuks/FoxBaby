# 🦊 FoxBaby

FoxBaby est une application de choix de prénom pour un bébé à naître, où les deux parents découvrent des prénoms sous forme de « Tinder ».

## Concept

- Chaque parent fait défiler des prénoms un par un.
- **Swipe à gauche** pour refuser un prénom.
- **Swipe à droite** pour valider un prénom.
- Quand les **deux parents valident le même prénom**, cela crée un **match** ! 🎉

## Fonctionnalités

- **Swipe de prénoms** : interface inspirée de Tinder pour valider ou refuser les prénoms.
- **Matchs** : un prénom validé par les deux parents devient un match.
- **Liste de favoris** : chaque parent peut consulter sa propre liste de prénoms favoris.
- **Filtre par sexe** : classer et filtrer les prénoms par sexe (garçon ou fille).

## Stack technique

- React + TypeScript
- Vite

## Démarrage

```bash
npm install
npm run dev
```
